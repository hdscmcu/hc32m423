平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本样例主要展示CTC通过外部高速时钟校准内部高速振荡器HRC。

说明：
1、首先设置CTC_CR1 TRMVAL为0x21，使内部高速时钟频率在16±0.16MHz范围外
2、然后通过外部高速时钟8MHz校准内部高速时钟16MHz，校准偏差为1.0%，目标频率为
   16±0.16MHz范围内

---------------------
串口助手

================================================================================
使用步骤
================================================================================
1）将引脚J4_PD3与示波器相连，该引脚输出HRC时钟信号;
2）连接DAP模块J9,J10
3）打开串口助手软件，配置端口如下参数
     波特率：115200
     数据位：8
     校验位：None
     停止位：1
4）重新编译，下载代码
5）观察示波器，查看波形频率为16±0.16MHz范围外
6）按测试板按键SW1
7）查看RGB LED灯状态
   若RGB绿色LED灯点亮，表示Trimming成功，查看串口助手打印信息
   若RGB红色LED灯点亮，表示Trimming失败
8）观察示波器，查看波形频率为16±0.16MHz范围内，符合预期
9）按测试板按键NRST，复位开发板
10）重复步骤5）~9），观察LED灯状态和串口助手打印信息

================================================================================
