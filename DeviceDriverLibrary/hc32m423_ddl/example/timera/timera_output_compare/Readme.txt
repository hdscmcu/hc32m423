平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本样例展示Timera的PWM输出及硬件触发启动/停止功能。

说明：
本样例设置Timera为三角波计数模式，Timera单元4的通道1配置为PWM输出模式并开启缓存
功能，通过按键SW1启动/停止Timera，通过按键SW2改变输出占空比，将示波器连接到P92
（通道1）观察输出波形。

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载程序功能，关闭IDE下载界面；
3）按键SW1启动Timera的PWM输出功能，观察LED_R常亮，PWM频率为50Hz；
4）将示波器连接到P92观察输出波形，反复点击按键SW2改变输出占空比；
5）观察示波器波形变化，再次按下SW1暂停Timera，停止PWM输出；
6）观察示波器没有波形输出，可反复执行3-5步骤。
