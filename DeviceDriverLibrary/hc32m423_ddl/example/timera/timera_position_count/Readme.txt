平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本样例展示Timera的2相正交编码位置计数功能。

说明：
本样例设置Timera时钟源为正交编码端口CLKA和CLKB输入，设置P74为端口CLKA、P73为端口
CLKB，配置端口CLKB为高电平时，端口CLKA采样到上升沿触发递加计数，配置端口CLKB为低
电平时，端口CLKA采样到上升沿触发递减计数，递加计数溢出则LED_G切换状态，递减计数
溢出则LED_R切换状态。

================================================================================
使用步骤
================================================================================
1）使用杜邦线连接P94和P74引脚，再连接P11和P73引脚；
2）打开工程并重新编译；
3）启动IDE的下载程序功能，关闭IDE下载界面；
4）此时CLKA输入波形的上升沿超前CLKB 1/4周期，观察LED_R闪烁；
5）按下按键SW1，此时CLKA输入波形的上升沿滞后CLKB 1/4周期，观察LED_G闪烁；
6）重复点击按键SW1，可循环执行步骤4-5。
