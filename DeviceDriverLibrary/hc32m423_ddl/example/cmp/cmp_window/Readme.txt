平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
本工程为CMP外设通过CMP2和CMP3两个通道实现窗口比较模式的样例，配置如下：
IVREF1为窗口低电压；
IVREF2为窗口高电压；
IVCMP2_0为输入比较电压；
通过VCOUT2引脚输出比较结果。

使用步骤
================================================================================
1）打开工程并重新编译。
2）芯片引脚IVCMP2_2（P47）作为输入比较电压,EVB板上已经有可调变阻器ADC可以调节电压;
3）芯片引脚IVREF1（P11）输入窗口低电压（如2.3V）
   芯片引脚IVREF2（P10）输入窗口高电压（如3.5V）
4）全速运行样例代码。
5）调节可调变阻器ADC改变输入比较电压值，通过示波器查看引脚VCOUT2（P23）变化。
   当比较电压<IVREF1，VCOUT2输出低；
   当比较电压>IVREF2，VCOUT2输出低；
   其他电压VCOUT2输出高。
