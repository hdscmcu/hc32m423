平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
本样例为CMP外设配置为普通比较模式时,选择IVREF1为参考电压，IVCMP1_0为比较电压，
比较器输出极性反向，通过VCOUT引脚输出比较结果的样例。

使用步骤
================================================================================
1）打开工程并重新编译。
2）芯片引脚P40连接输入比较电压（<VCC）。
   采用EVB板上ADC分压电阻产生的ADC_IN（P47）作为基准电压，调节ADC变阻器使P47输出
   2V电压，连接P47与IVCMP1_0（P11）。
3）全速运行样例代码。
4）调节输入比较电压值，用示波器查看VCOUT（P24）输出。
5）当比较电压>2V，VCOUT输出低；
   当比较电压<2V，VCOUT输出高。
