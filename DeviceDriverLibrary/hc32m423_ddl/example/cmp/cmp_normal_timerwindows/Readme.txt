平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
本样例为CMP3外设配置为普通比较模式的timer window功能时,选择芯片内部8 bit DAC为参
考电压（配置为VCCA/2），IVCMP3_0为比较电压，选择TIM4_OUH信号为timer window功能的
控制信号,通过VCOUT引脚输出比较结果的样例。

使用步骤
================================================================================
1）打开工程并重新编译。
2）芯片引脚IVCMP3_0（P42）连接输入比较电压(<VCC),
   TIM4_1_OUH（P71）接入示波器查看PWM输出波形。
   VCOUT3（PA2）接入示波器查看VCOUT输出波形。
3）全速运行样例代码。
4）调节输入比较电压值，查看VCOUT输出波形。
5）PWM输出高时，CMP输出有效，如下：
     当比较电压>VCCA/2，VCOUT输出高；
     当比较电压<VCCA/2，VCOUT输出低。
   PWM输出低时，CMP输出无效，VCOUT默认为高电平。
