平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
本样例为时钟频率测量，用XTAL(8MHz) 8192分频或者1024分频作为参考基准，测量：
HRC     (16MHz)
LRC     (32K)
SWDTLRC (16K)
PCLK1   (8MHz,系统时钟为MRC_8MHz)
MRC     (8MHz)
PLLP    (80MHz，VCO_160MHz div2)

================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV_M423_LQ64_V10

辅助工具:
---------------------
无

辅助软件:
---------------------
串口调试助手（115200bps，8N1）

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
   如需要对SWDTLRC时钟进行测量,需要打开main.c中的宏定义
   #define FCM_SWDTLRC_ON      (1UL)
   并且修改hc32m423_icg.h文件中宏定义的值
   #define ICG0_RB_SWDT_AUTST  (ICG_SWDT_RST_START)
2）启动IDE的下载和调试功能，全速运行；
3）打开串口调试助手，选择识别的COM口，设置波特率115200bps，8N1；
4）短按按键SW1，被测时钟频率将被打印至串口终端。
