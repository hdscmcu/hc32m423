平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
本样例为flash连续编程，全擦除。

使用步骤
================================================================================
此样例需运行在RAM演示，IAR配置如下；
1）打开工程，options->Linker->config，更改linker configuration file，选择
   HC32M423_RAM.icf；
2）options->Debugger->Download，取消勾选“use flash loader(s)”
3）重新编译；
4）启动IDE的下载和调试功能,main.c的171行，174行，177行打上断点；
5）全速运行，可断点查看memory窗口变化：扇区20和140擦除->扇区20和140连续编程
->flash全擦除。
