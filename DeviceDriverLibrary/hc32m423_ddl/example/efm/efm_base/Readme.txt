平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
本样例为EFM扇区擦除,半字编程，字编程，编程正确亮绿灯，不正确熄灭绿灯。
本样例以扇区80为例。

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能，main.c的132行，138行，144行，152行打上断点；
3）打开IDE的memory窗口，观察0x0000A000地址；
4）运行程序至断点处可观察到地址0x0000A000有数据写入，并且绿灯常亮，编程正确。
