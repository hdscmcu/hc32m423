平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本例程实现了SPI模块工作在主机模式下的配置和基本应用。

说明：
将SPI_FUNC_TEST_SLAVE定义位非0，可用于测试例程spi_slave和spi_dam。

================================================================================
注意
================================================================================
1、SPI工作在4线模式时，每个片选信号有效周期内，只发送或接收一个单位的数据（一个
   字节或一个半字）。
2、有的外围SPI器件，需要在一个片选信号有效周期内，收发多个数据，此时需要将SPI设置
   为3线模式，并自定义一个引脚作为片选引脚，具体用法请参考例程spi_w25q54。
3、如果使能奇偶校验功能，有效数据的位数减少1位，最后一个发送的位即是奇偶校验位，即，
   如果先发送MSB，那么LSB即为奇偶校验位，反之MSB是奇偶校验位。
