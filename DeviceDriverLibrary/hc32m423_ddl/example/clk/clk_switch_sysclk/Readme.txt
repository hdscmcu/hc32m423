平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
本样例主要是系统时钟切换，并输出系统时钟的16分频。

无

================================================================================
使用步骤
================================================================================
1）连接P70与示波器,用于观察MCO的输出的系统时钟的16分频。
2）打开工程并重新编译；
   可以通过对宏定义EXAMPLE_PLL_SRC_XTAL的修改来配置PLL时钟源是选择XTAL或者HRC。
3）启动IDE的下载和调试功能，全速运行；
4）通过按键SW1切换系统时钟频率；
5）查看示波器,可观察到波形及频率变化，
    代码运行后系统默认以MRC为系统时钟，MCO输出8M/16;
    短按按键SW1，系统时钟切换为HRC，MCO输出F(HRC)/16;
    短按按键SW1，系统时钟切换为LRC，MCO输出32K/16;
    短按按键SW1，系统时钟切换为XTAL，MCO输出8M/16;
    短按按键SW1，系统时钟切换为PLL，MCO输出80M/16或者60M/16。
