平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本样例主要展示EMB模块的电压比较器比较结果制动功能。

说明：
EMB检测电压比较器比较结果标志位，
1、比较器比较结果标志位置起时，PWM输出指定电平，产生EMB中断，执行EMB中断服务回调
   函数
2、EMB中断回调函数内，然后清除比较器比较结果标志位和EMB状态，PWM正常输出

================================================================================
使用步骤
================================================================================
1）测试板J6-P71(TIM4_1_OUH)、J6-P74(TIM4_1_OUL)引脚与示波器连接，J7-P40与J4-P02
   连接
2）编译、下载并运行
3）观察示波器，显示J6-P71、J6-P74通道周期脉冲（1Hz）
4）按下按键SW1，不释放
5）观察示波器，显示J6-P71、J6-P74通道电平为低
6）释放按键SW1
7）观察示波器，显示J6-P71、J6-P74通道周期脉冲（1Hz）
8）重复步骤3）~步骤7）
